[编程题] 删数
========
- - -

有一个数组a[N]顺序存放0-N，要求没隔两个数删掉一个数，到末尾时循环至开头继续进行，求最后一个被删掉的数的原始下标位置。以8个数(N=7)为例:｛0，1，2，3，4，5，6，7｝，0->1->2(删除)->3->4->5(删除)->6->7->0(删除),如此循环直到最后一个数被删除。

输入描述:
=====

>每组数据为一行一个整数n(小于等于1000)，为数组成员数,如100，则对a[999]进行计算。


输出描述:
=====

>一行输出最后一个被删掉的数的原始下标位置。

输入例子:
=====

>8
输出例子:
=====

>6

``` C++
#include <iostream>
using namespace std;
int main()
{
    int n, p[1000];
  
    while (cin >> n)
    {
        for (int i = 0; i < n; i++)
            p[i] = 1;
 
        int index = 0, count = 0;
        for (int i = 0; count < n - 1 ; i++)
        {
            if (p[i % n] == 0)
                continue;
            if ((index + 1) % 3 == 0)
            {   
                p[i % n] = 0;
                count += 1;
            }
            index++;
        }
         
        for (int i = 0; i < n; i++)
        { 
            if (p[i])
            {cout << i << endl; break;}
        }
    }
    return 0;
}
```
